# 🏝️ Ilhas de Aprendizado (HTML + CSS + JS + JSON)

Projeto **estático** (sem backend) para simular uma plataforma de trilhas por setor, onde **cada curso concluído desbloqueia um item visual na ilha** (ex: concluir “Contabilidade” faz aparecer um **💲** na ilha de Finanças).

Perfeito para **apresentação rápida** e deploy simples na **Vercel**.

---

## ✅ O que este projeto entrega

- **Ilhas por setor** (ex: Finanças, RH, TI)
- Cada ilha tem **5 cursos**
- Ao concluir um curso, um **ícone aparece em uma posição específica da ilha**
- **2 perfis de usuário**
  - **Estagiário**: vê todas as ilhas e pode concluir cursos
  - **Admin**: vê **somente usuários do seu setor** e o progresso deles
- Progresso salvo no **localStorage** (não perde ao recarregar)
